# Local Job Posting Website

A frontend job listing platform that allows users to browse local job opportunities, view hiring companies, and apply for jobs.

## Features
- Job listings with filters
- Company showcase
- Job application form
- Responsive design

## Technologies Used
- HTML5
- CSS3
- Responsive Design (Flexbox & Grid)

## Pages
- Home
- Jobs
- Companies
- Apply
- About
